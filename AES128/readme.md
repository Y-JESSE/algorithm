<h1 align="center">AES-128使用说明</h1>


## 示例
```
int main(void)
{
    unsigned char data[] = {0x00, 0x01, 0x02, 0x03, 0x04, 0x05, 0x06, 0x07, 0x08, 0x09, 0x0a, 0x0b, 0x0c, 0x0d, 0x0e, 0x0f};
    unsigned char key1[] = {0x00, 0x01, 0x02, 0x03, 0x04, 0x05, 0x06, 0x07, 0x08, 0x09, 0x0a, 0x0b, 0x0c, 0x0d, 0x0e, 0x0f};
    unsigned char key2[] = {0x00, 0x01, 0x02, 0x03, 0x04, 0x05, 0x06, 0x07, 0x08, 0x09, 0x0a, 0x0b, 0x0c, 0x0d, 0x0e, 0x0f};

    aes128_enc_dec(data, key1, 0);
    printf("encode data:");
    for(int i=0; i<16; i++){
        printf("0x%02x ",data[i]);
    }
    printf("\r\n");
    
    printf("decode data:");
    aes128_enc_dec(data, key2, 1);
    for(int i=0; i<16; i++){
        printf("0x%02x ",data[i]);
    }
    printf("\r\n");
}

运行结果：
encode data:0x0a 0x94 0x0b 0xb5 0x41 0x6e 0xf0 0x45 0xf1 0xc3 0x94 0x58 0xc6 0x53 0xea 0x5a 
decode data:0x00 0x01 0x02 0x03 0x04 0x05 0x06 0x07 0x08 0x09 0x0a 0x0b 0x0c 0x0d 0x0e 0x0f 
```
